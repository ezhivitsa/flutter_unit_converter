# Unit converter

Unit converter created with flutter.

## Features

* Convert distance values:
  * Meter
  * Kilometer
  * Centimeter
  * Millimeter
  * Mile
  * Yard
  * Foot

* Convert weight values:
  * Kilogram
  * Gram
  * Miligram
  * Metric Ton
  * Long Ton
  * Pound

* Convert time values:
  * Second
  * Minute
  * Hour
  * Day
  * Week
